Sécurité des Réseaux

Introduction
La sécurité des réseaux protège les données et les ordinateurs contre les attaques et les intrusions.
Les Menaces
  • Virus et logiciels dangereux
  • Attaques qui bloquent le réseau
  • Emails trompeurs pour voler vos informations
  • Intrusions dans le réseau
Comment se protéger
Pare-feu :
Bloque les accès non autorisés.

Antivirus :
Élimine les virus et logiciels dangereux.

Chiffrement :
Rend les données illisibles pour les intrus.

Contrôle des accès :
Vérifie qui peut utiliser le réseau et quoi faire.
Exemple de règles
Règle Description
Mot de passe fort Créer un mot de passe difficile et le changer souvent
VPN Connexion sécurisée pour les utilisateurs à distance
Surveillance du réseau Vérifier régulièrement le réseau pour détecter les problèmes
